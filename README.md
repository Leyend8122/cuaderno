# 📘 Avances en Clases de Python

Este repositorio contiene los apuntes, ejercicios y proyectos desarrollados durante el curso de Python.  
El objetivo es documentar el progreso y servir como referencia para futuras prácticas.

---

## 🚀 Objetivos del curso
- Comprender los fundamentos de Python.
- Desarrollar programas básicos y estructurados.
- Aplicar conceptos de programación orientada a objetos.
- Manejar librerías comunes para análisis y manipulación de datos.
